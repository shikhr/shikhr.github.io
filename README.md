# Portfolio Site

A modern, responsive portfolio website built with Next.js, React, and Material-UI.

## Tech Stack

- **Framework:** Next.js 16
- **UI Library:** React 19
- **Styling:** Material-UI, Tailwind CSS
- **Animations:** Framer Motion
- **Language:** TypeScript

## Getting Started

Install dependencies:

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the site.

## Build

Create a production build:

```bash
pnpm build
```

## Deployment

This site is deployed on GitHub Pages at [shikhr.github.io](https://shikhr.github.io).
